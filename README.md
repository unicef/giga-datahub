# Giga Datahub

## Prerequisites

- [ ] Kubernetes
- [ ] Task

## Development

1. In the `k8s` directory, copy contents of `secrets.example.yaml` into a new file
   named `secrets.yaml`.
2. Fill in the missing values.
3. Run `task` to install the K8s & Helm items.
4. Open the Datahub UI at `localhost:9002`.
5. Go wild.
